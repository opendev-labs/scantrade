System Overview

The Governed Trading System is a high-performance, semi-automated trading platform designed for paper trading and institutional-grade simulation. It employs a decoupled architecture with a FastAPI backend and a Next.js frontend, ensuring scalability and real-time responsiveness.

Backend (FastAPI)

  • Asynchronous trading engine
  • Integrated risk management
  • SQLite database with SQLAlchemy
  • Strategy execution & scanner logic

Frontend (Next.js)

  • Real-time dashboard visualization
  • Module-based control plane
  • Responsive design with Tailwind CSS
  • Interactive strategy management

Key Components

Trading Engine

Orchestrates the lifecycle of market data fetching, scanner execution, and bot signal processing every 60 seconds.

Risk Manager

Enforces multi-layer governance including max drawdown, portfolio exposure, and per-bot risk limits.

Scanner System

Decoupled modules that analyze market state, trend alignment, and volatility compression.
